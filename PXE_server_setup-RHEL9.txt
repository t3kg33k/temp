The PXE/DHCP server will need the following:
DHCP server package
TFTP server package
NFS Utils package for NFS server
RHEL 9 ISO
Kickstart file

1. Install the following services:

	dhcp-server
	tftp-server
	nfs-utils
	
	dnf install dhcp-server -y
	dnf install tftp-server -y
	dnf install nfs-utils -y
	
2. Start and enable the services.

	systemctl enable --now dhcpd
	systemctl enable --now tftp.socket
	systemctl enable --now nfs-server.service

3. Make the following directories for NFS shares for RHEL 9 source and Kickstart file.
	
	mkdir /ks
	mkdir /rhel9iso
	
4. Mount the RHEL 9 ISO and copy to /rhel9iso.

	mkdir /mnt/rhel9iso
	mount -o loop,ro -t iso9660 rhel-9.7-x86_64-dvd.iso /mnt/rhel9iso
	cp -r /mnt/rhel9iso /  # NOTE: ensure .treeinfo from installation source gets copied to destination directory /rhel9iso
	
5. Configure NFS server (or copy over existing) /etc/exports with the following shares:

	/ks/ *
	/rhel9iso/ *
	
	and update with the following command:
	
	exportfs -r
	
6. Configure DHCP server (or copy over existing) /etc/dhcpd.conf with the following:

#
# DHCP Server Configuration file.
#   see /usr/share/doc/dhcp-server/dhcpd.conf.example
#   see dhcpd.conf(5) man page
#

option architecture-type code 93 = unsigned integer 16;

subnet 10.x.x.x netmask 255.255.255.0 {
    range 10.x.x.x 10.x.x.x;
    option routers 10.x.x.x;
#    option domain-name-servers x.x.x.x;
    next-server 10.x.x.x;

    class "pxeclients" {
        match if substring (option vendor-class-identifier, 0, 9) = "PXEClient";

        # Serve the correct boot loader based on client architecture
        if option architecture-type = 00:07 {
            filename "redhat/EFI/BOOT/BOOTX64.EFI";
        } else {
            filename "pxelinux.0";
        }
    }
}
host tbtst004 {
        hardware ethernet xx:xx:xx:xx:xx:xx;
        fixed-address 10.x.x.x;
        option host-name "tbtst004";
}

7. Make the following directories in TFTP directory for boot configs and copy the necessary files from the installation source.

	mkdir /var/lib/tftpboot/redhat
	mkdir -p /var/lib/tftpboot/images/RHEL-9
	
	cp -r /rhel9iso/EFI /var/lib/tftpboot/redhat/
	cp /rhel9iso/images/pxeboot/{vmlinuz,initrd.img} /var/lib/tftpboot/images/RHEL-9/
	
	chmod -R 755 /var/lib/tftpboot/redhat/
	
8. Update /var/lib/tftpboot/redhat/EFI/BOOT/grub.cfg with the following:

set timeout=2
menuentry 'RHEL 9' {
	linux images/RHEL-9/vmlinuz ip=dhcp inst.repo=nfs:10.x.x.x:/rhel9iso inst.ks=nfs:10.x.x.x:/ks/ks.cfg
	initrd images/RHEL-9/initrd.img
}
